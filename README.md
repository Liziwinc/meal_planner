# Конструктор рациона питания (Meal Planner)

Простой и эффективный инструмент для тех, кто хочет контролировать своё питание. Приложение позволяет хранить базу блюд с их ингредиентами, составлять план питания на произвольное количество дней и получать суммарный список продуктов, которые нужно купить.

## Основные возможности

* **Управление базой блюд:** добавление новых рецептов с указанием типа (завтрак/обед), калорийности и состава.
* **Гибкое планирование:** составление рациона на любое количество дней (от 1 до 30).
* **Автоматический расчет:** приложение суммирует калории за день и учитывает количество порций.
* **Список покупок:** генерация общего перечня ингредиентов для всех выбранных блюд с учетом множителя порций.
* **Кроссплатформенность:** поддержка Windows, Linux и macOS.

## Технологический стек

* **Язык:** Python 3.10+
* **База данных:** SQLite
* **Сборка:** PyInstaller 

## Структура проекта

* `main.py` — основной файл приложения, отвечающий за логику меню и интерфейс.
* `database.py` — функции для работы с базой данных (создание таблиц, CRUD операции).
* `utils.py` — вспомогательные утилиты для ввода данных и форматирования таблиц.
* `fill_db.py` — скрипт для первичного наполнения базы данных готовыми наборами блюд (25+ рецептов).
* `.github/workflows/build.yml` — конфигурация CI/CD для автоматической сборки `.exe` и бинарных файлов.

## Установка и запуск

### Запуск из исходного кода
1. Убедитесь, что у вас установлен Python 3.10 или выше.
2. Склонируйте репозиторий.
3. (Опционально) Заполните базу данных начальными данными:
   ```bash
   python fill_db.py
   ```
4. Запустите приложение:
   ```bash
   python main.py
   ```

### Использование готовых сборок
Вы можете скачать скомпилированный файл для вашей ОС из раздела **Releases** (генерируется автоматически через GitHub Actions). В этом случае устанавливать Python не требуется.

## Инструкция по работе

1.  **Добавление блюд:** Если база пуста, начните с пункта "1" в главном меню или запустите `fill_db.py`.
2.  **Составление рациона:**
    * Выберите пункт "3".
    * Укажите количество дней и количество порций (например, если вы готовите на двоих, введите 2).
    * Для каждого приема пищи выберите блюдо из предложенного списка.
3.  **Результат:** Программа выведет план по дням с калорийностью и итоговый список покупок в граммах.

## Хранение данных
База данных создается автоматически в домашней директории пользователя под именем `meal_planner.db`. Это гарантирует сохранность ваших данных даже при обновлении версии программы.
